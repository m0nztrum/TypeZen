# TypeZen

A minimal typing practice web application.

## Front End

-   React.js and Tailwind for styling
    -   NavBar
        -   App Name
    -   Input Area
        -   Highlighted whiles typing (wrong, current, correct)
        -   Focus bluring
    -   Stats
        -   WPM
        -   Accuracy
        -   Mistakes

## Backend

-   Supabase (BAAS)
    -   Authentication
    -   Database (For Leaderboards, Tracking progress)

## Deployment

-   Deployed on vercel
